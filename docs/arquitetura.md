
# Arquitetura do IA-JOMARO

## Visão Geral

O IA-JOMARO é um laboratório pessoal de Inteligência Artificial.

O objetivo é construir um sistema de IA modular, baseado em tecnologias open source, que possa utilizar diferentes modelos de linguagem (LLMs), incluindo futuramente o AMÁLIA.

O sistema deverá crescer de forma organizada, permitindo adicionar novos agentes, novas fontes de conhecimento e novas funcionalidades sem alterar a estrutura principal.

---

# Objetivos

- Aprender Inteligência Artificial de forma prática.
- Desenvolver assistentes inteligentes em português europeu.
- Experimentar diferentes modelos de IA.
- Criar uma arquitetura modular e reutilizável.
- Integrar o modelo AMÁLIA quando for adequado.
