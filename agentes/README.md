# Agentes

Esta pasta contém os agentes do sistema IA-JOMARO.

## O que é um agente?
Um agente é uma “personalidade especializada” da IA, com um papel específico.

## Objetivo
Criar diferentes assistentes dentro do mesmo sistema, cada um focado numa tarefa.

## Exemplos de agentes futuros
- Programador: ajuda a escrever e corrigir código
- Escritor: ajuda a criar textos e relatórios
- Professor: explica conceitos de forma simples
- Investigador: analisa informação e faz sínteses
- Produtividade: ajuda a organizar tarefas e ideias
