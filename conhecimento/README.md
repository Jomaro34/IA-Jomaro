# Conhecimento

Esta pasta funciona como a memória do sistema IA-JOMARO.

## Objetivo
Guardar informação que pode ser usada pela IA para responder melhor e de forma contextualizada.

## Tipos de conteúdo
- Documentos PDF
- Notas pessoais
- Textos de referência
- Informação técnica
- Materiais de estudo

## Uso futuro
Esta informação será usada para:
- Sistemas de pesquisa (RAG)
- Respostas mais precisas
- Assistentes personalizados
- Integração com modelos como AMÁLIA
